Handwritten Import Design Options

These HTML wireframes compare the two Phase 3 directions for Recetas handwritten recipe import. Both preserve the required secondary-tab pattern and the existing preview/edit flow, but they differ in how strongly the handwritten source workflow is framed.

Option A

Guided Split-Mode

One active mode at a time with a simple handwritten upload block.
Lowest-risk extension of the current import page.
Most conservative fit with current Recetas structure.

Option B

Source-First Workspace
